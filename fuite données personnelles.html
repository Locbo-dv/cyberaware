Fuite / violation de données personnelles

Une fuite de données (ou violation de données) correspond à un accès, une divulgation ou un vol non autorisé d’informations personnelles détenues par un service (site web, appli, entreprise, administration…).

← Retour aux menaces Comment se protéger

Comment ça arrive ?

Une fuite de données peut se produire après un piratage, une erreur de configuration, un mot de passe trop faible, ou encore une faille sur un service en ligne. Les informations exposées peuvent inclure :

  • nom, prénom, date de naissance
  • adresse, email, numéro de téléphone
  • identifiants / mots de passe (parfois chiffrés, parfois non)
  • infos bancaires (dans certains cas)

Quels sont les risques ?

Usurpation d’identité

Création de comptes ou démarches frauduleuses avec tes informations.

Piratage de comptes

Test de tes mots de passe sur d’autres sites (“credential stuffing”).

Phishing ciblé

Arnaques plus crédibles (email/SMS) car l’attaquant connaît tes infos.

Fraude financière

Achats, tentatives d’accès bancaire ou demandes de paiement.

Que faire immédiatement ?

  1. Identifier le service concerné (site/appli/entreprise) et lire leur message d’alerte.
  2. Changer le mot de passe du compte touché (et des comptes où tu utilisais le même).
  3. Activer la double authentification (2FA) quand c’est possible.
  4. Surveiller tes comptes (emails, connexions, banque, achats, messages suspects).
  5. Se méfier des emails/SMS qui parlent de “sécurité” : vérifie toujours l’expéditeur.

Bonnes pratiques pour limiter l’impact

  • Utilise des mots de passe uniques (un par site).
  • Privilégie un gestionnaire de mots de passe.
  • Active la 2FA sur les comptes importants (mail, réseaux sociaux, banque, jeux…).
  • Vérifie régulièrement si tes identifiants ont fuité.
  • Ne clique pas sur les liens d’un message “urgent” : passe par le site officiel.
